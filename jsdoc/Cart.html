Class: Cart

Cart()

Represents the cart. It contains an array of CartItems. This array can be saved/loaded to/from localstorage. There can be serveral items with the same productId, as long as they have different colors. Only one item with the same productId and Color can exist in the array. The property : quantity of this item can go from 1 to infinity.

Constructor

new Cart()

Create a Cart instance. If the cart has been saved in another page : load the array from localstorage. If there is no cart data in localstorage : creates a empty array.
